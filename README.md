# Ahmiyat Blockchain

A fast and scalable blockchain for decentralized transactions.

## Prerequisites
- g++, libssl-dev, libleveldb-dev, libcurl4-openssl-dev, libmicrohttpd-dev
- IPFS (install manually in Codespaces using binary method)
